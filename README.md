# BankApp

A web application that emulates interactions with an ATM or with a teller at a bank. 

This typescript project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.8.

## Installation

Before anything, make sure you at least have:

* Angular-cli 6.0.8
* Node 8.11.2
* NPM 5.6.0

Next, run the command `git clone https://github.com/awstin17/Bank-Application.git` to clone to a local repository.

## Development server

Enter the Bank-Application directory.

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
